# crab-cryptor

## About

A simple file cryptor written in Rust.

## Install

```bash
cargo install crab-cryptor
```

## Getting Started

### Encrypt

```bash
crab
```

```bash
crab v4.0.1
Author: xl_g <lr_kkr@outlook.com>
A secure file cryptor

> Choose function: encrypt
> Work directory: data
> Encryption password:
```

### Decrypt

```bash
crab
```

```bash
crab v4.0.1
```
