# crab

## About

A simple file cryptor written in Rust.

## Install

```bash
cargo install crab-cryptor
```

## Getting Started

### Encrypt

```bash
crab -e superKey -p path/to/
```

### Decrypt

```bash
crab -d superKey -p path/to/
```
